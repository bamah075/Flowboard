## LinkedIn — Long-form post

**We built a free Kanban board for website projects. No login. No subscription. Just open it.**

After managing dozens of website builds across spreadsheets, sticky notes, and overpriced tools — we got tired of paying for project trackers that were 10× more complex than the job needed.

So we built FlowBoard.

It's a single HTML file. Open it in any browser and you're inside a fully functional drag-and-drop Kanban board — pre-loaded with the stages every website project needs: Backlog → Planning → Design → In Dev → Review → Live.

Here's what makes it different:

→ **No login** — share the file, it just works
→ **Auto-saves** — everything persists in your browser, no server needed
→ **6 starter templates** — Website Build, Campaign, Product Launch, Client Project, Content Calendar
→ **Export/Import** — back up your boards as JSON anytime
→ **Multi-project dashboard** — manage multiple client builds in one place
→ **Claude plugin** — ask Claude to generate a custom board pre-loaded with your pages and team

It's completely free. The source is on GitHub so you can fork it, host it, or customise it however you need.

Live: flowboard.vel-ai-au.com
GitHub: github.com/bamah075/Flowboard
Claude plugin: /plugin marketplace add bamah075/Flowboard

If you're building websites for clients and you've been tracking progress in a spreadsheet — try this instead.
